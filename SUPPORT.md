# Support

## Getting Help

For usage questions, setup help, or debugging assistance:

- open a GitHub issue with details
- include logs, config snippet (without secrets), and repro steps

## Bug Reports

When reporting bugs, include:

- expected behavior
- actual behavior
- environment (OS, Rust version, Bun version)
- backend/frontend command output

## Security Issues

Do not report security issues publicly.

See [`SECURITY.md`](./SECURITY.md) for responsible disclosure guidance.
